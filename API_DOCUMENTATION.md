 # 炼书 Todo 插件公共接口与组件文档
 
 ## 概览
 
 本仓库当前仅包含静态隐私政策页面 `index.html`，用于向用户展示《炼书 Todo》插件的隐私政策说明。页面不内置任何前端框架、构建脚本或动态逻辑，也未导出可供其他模块调用的 JavaScript/TypeScript 函数，因此不存在独立的 API 或服务端接口。
 
 ## 公共组件与结构
 
 - **`index.html`**：单页隐私政策文档，直接以原生 HTML/CSS 编写，无外部依赖。
   - **页面标题区域**：包含 `<h1>` 主标题和生效日期段落。
   - **隐私条款章节**：通过多个 `<h2>` 标题划分章节（如“信息收集”“数据存储”“数据安全”等），每个章节对应一段 `<p>` 描述。
   - **联系方式**：在“联系我们”章节内展示邮箱，用于接收用户反馈。

     <h1>隐私政策</h1>

 ## 使用说明
 
 - **本地预览**：直接在浏览器中打开 `index.html` 即可查看完整内容。
 - **部署到静态站点**：可将文件上传至任意静态托管平台（如 GitHub Pages、Vercel、Netlify），或通过任意静态服务器（例如 `npx serve`）进行本地托管。
 - **集成到插件或产品**：
   1. 在插件设置或帮助菜单中提供跳转链接，指向托管的隐私政策地址。
   2. 或在插件内部嵌入 WebView/iframe，直接加载此页面。
 
 ## 自定义与扩展示例
 
 ### 更新生效日期
 
 1. 打开 `index.html`。
 2. 将 `<p>生效日期：2025.02.18</p>` 更新为最新日期格式（建议保持 `YYYY.MM.DD`）。
 
 ### 添加新章节
 
 若隐私政策需要新增条款，可在合适位置插入新的 `<h2>` 与 `<p>`：
 
 ```html
 <h2>8. 用户权利</h2>
 <p>您可随时通过联系我们的方式，提出对隐私政策的疑问或更新请求。</p>
 ```
 
 ### 国际化支持
 
 如果需要多语言版本，建议复制 `index.html`，并在 `<html lang="...">` 与文案中使用目标语言。例如，为英文版创建 `index-en.html`：
 
 ```html
 <!DOCTYPE html>
 <html lang="en">
   <!-- 其余结构与中文版本保持一致，替换为英文文案 -->
 </html>
 ```
